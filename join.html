In SQL, you have several options to concatenate columns:

  1. The || operator (in some databases like PostgreSQL):
    2. SELECT first_name || ' ' || last_name AS full_name FROM users;
  2. The + operator (in SQL Server):
    3. SELECT first_name + ' ' + last_name AS full_name FROM users;
  3. The CONCAT function (standard SQL):
    4. SELECT CONCAT(first_name, ' ', last_name) AS full_name FROM users;

For MySQL specifically, you can use:

SELECT CONCAT_WS(' ', first_name, last_name) AS full_name FROM users;

The CONCAT_WS function automatically handles NULL values and adds the separator between non-NULL values.
